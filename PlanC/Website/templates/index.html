Live ECG plot for the patient Alhassan Jawad

The upper figure shows a lead1 measurement while the bottom subplot shows a lead2 measurement. Both measurements are taken from the patient in a resting space.

The used materials are:

  • AD8232 ECG heart monitor
    • Takes the ECG voltages from the patient
    • Amplifies the voltages to be between 0 and 3.3V
  • Arduino Nano RP2040 Connect
    • Reads the voltages from the AD8232
    • Converts the voltages to digital values
    • Acts as a sender unit to send the data via bluetooth
  • Arduino Nano 33 BLE
    • Acts as a receiver unit to receive the data via bluetooth
    • Prints the voltages in real time in the serial plotter
    • Starts a timestamp when the first data is received prints the timestamp using the Millis() function
  • Python 3.11.1
    • Used to "spy" on the serial port to receive the data (using Pyserial 3.5)
    • Plots the data in real time using the matplotlib library (using Matplotlib 3.6.9)
